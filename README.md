# ChromaView: Palette Playground

A web-based tool for generating, simulating, and exporting color palettes.  
Built with HTML, CSS, and JavaScript, this project helps designers explore colors, test accessibility, and copy HEX codes easily.

## Features
- **Generate Again**: Refresh the palette instantly using spacebar.
- **Color Blindness Simulation**: Preview palettes under Protanopia, Deuteranopia, Tritanopia, and Achromatopsia.
- **Copy HEX Codes**: Export your palette values with one click.
